## Quick Code Usage

```python
from vedic_astrology_system_integrated import VedicAstrologySystem, ChartData, PlanetPosition

# Initialize
system = VedicAstrologySystem()

# Create chart
chart = ChartData(
    person_name="John Doe",
    lagna_sign=1,      # Aries
    moon_sign=5,       # Leo
    sun_sign=3,        # Gemini
    moon_nakshatra="Magha"
)

# Add planet
chart.planets["Venus"] = PlanetPosition(
    planet="Venus",
    sign_number=2,
    sign_name="Taurus",
    house_number=12
)

# Analyze
report = system.analyze_individual_chart(chart)
print(report["detailed_text_report"])
```

## Common Tasks

### Check Marriage Promise
```python
from vedic_astrology_system_integrated import MarriageAnalyzer, AstrologyRulesEngine

engine = AstrologyRulesEngine()
analyzer = MarriageAnalyzer(chart, engine)
promise, factors = analyzer.check_marriage_promise()
print(f"Marriage: {'YES' if promise else 'NO'}")
```

### Get Compatibility Score
```python
compat = system.analyze_compatibility(chart_a, chart_b)
print(f"Score: {compat['kuta_milan_scores']['percentage']:.1f}%")
```

### Derive Spouse Name (Nadi)
```python
from vedic_astrology_system_integrated import NadiMarriageAnalyzer

nadi = NadiMarriageAnalyzer(chart, engine)
result = nadi.derive_spouse_name_seed()
print(f"Syllables: {result['candidate_syllables']}")
```
